## Crosshair Tutorial
1. Go to https://alimg.github.io/crosshair-for-survivio/
2. Create your crosshair or upload an svg
3. Drag the red "CROSSHAIR FOR SURVIVIO" cell to the URL bar
4. Copy the code and paste it into the menu
