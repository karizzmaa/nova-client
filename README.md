## üöÄ Getting Started

### Requirements
1. A userscript manager (like **Tampermonkey** / **Violentmonkey**) installed in your browser.
2. A modern browser (Chrome / Firefox / Edge).

### Installation
1. Install a userscript manager from its official site.
2. Download the [userscript](https://raw.githubusercontent.com/karizzmaa/nova-client/main/dist/userscript.js). 
3. Click **Install** in your userscript manager prompt.
4. Open **Survev.io** and enjoy!

## üïπÔ∏è How to Use
Once the script is installed and Survev.io is running:
- Press **Right Shift** to open the Client menu.
- Toggle features, customize HUDs, and adjust settings in-game.
